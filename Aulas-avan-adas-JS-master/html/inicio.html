Forma de Armazena dados em JavaScript

No JavaScript existe várias formas de armazenar dados o dado pode ser Constante ou Variável

As constantes são declarados pela palavra reservada const

const teste4 = "Eu sou imutável" ;


As variáveis são declaradas pelas palavras deservadas var e let

var teste2 = "Minha String" ;


let teste1 = "Minha String" ;


Abra o console do navegador para ver os resultados.
