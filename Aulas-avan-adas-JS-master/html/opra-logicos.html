Os operadores lógicos em JavaScript são usados para "combinar condições ou expressões booleanas (verdadeiras ou falsas) e retornar um valor booleano como resultado".


Aqui estão os principais operadores lógicos são:


  • Operador E ( &&): Se Ambas as condições forem verdadeiras, o resultado será verdadeiro, caso contrário sempre será falso.

    • Exemplo:

    let x = 5;

    let y = 10;
    console.log(x > 3 && y < 15); // true (ambas as condições são verdadeiras)


  • Operador ou ( || ): Retorna true se pelo menos uma das condições for verdadeira.

    • Exemplo:

    let x = 5;

    let y = 10;
    console.log(x > 3 || y < 9); // true (a primeira condição é verdadeira)


  • Operador de Negação ( ! ): Inverte o valor booleano. Se a expressão for true, torna-se false e vice-versa.

    • Exemplo:

    let isTrue = true;
    console.log(!isTrue) // Resultado: false
